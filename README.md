# Python Tic Tac Toe

Just a fun tic-tac-toe game made with Python

![image](https://github.com/JamieBurridge/python-tic-tac-toe/assets/80159413/21e94b39-a099-443f-a0c0-f46b2d20328b)


### How to play

Dependencies:
- python3
- python3-venv
- pip3

Make the executable

```
make executable
```

Play the game

```
./dist/python-tic-tac-toe/python-tic-tac-toe
```
